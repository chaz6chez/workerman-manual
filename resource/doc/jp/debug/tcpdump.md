# ネットワークパケットキャプチャ

以下の例では、```tcpdump```を使用して```workerman-chat```アプリケーションが```websocket```を介して送受信するデータを表示します。```workerman-chat```の例では、サーバーは```7272```ポートを使用して```websocket```サービスを提供しているため、```7272```ポートでのデータパケットをキャプチャします。

1. *コマンドを実行* ```tcpdump -Ans 4096 -iany port 7272```

2. ブラウザのアドレスバーに```http://127.0.0.1:55151```を入力します。

3. ニックネームを入力します。```mynick```

4. メッセージボックスに```hi, all ！```と入力します。

*これまでのログインと発言に関するすべてのリクエストは、合わせて2つのブラウザクライアントで行われています。*

*パケットデータ中の```[S]```は```SYN```リクエスト（接続要求）を表し、```[.]```は```ACK```応答で、要求の相手側が既に受信したことを示します。[```P```]はデータ送信を示し、[P.]は[P] + [.]*

*ポート経由で送信されたデータがバイナリデータである場合は、16進数で表示することができます* ```tcpdump -XAns 4096 -iany port 7272```
